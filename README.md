# 劇団フルーツバスケット Web広告戦略 (2026年1月-3月)

このリポジトリは、2026年3月の入団募集プロジェクトに向けたWeb広告戦略の計画と進捗を管理するためのものです。

## プロジェクト概要
- **目的**: 2026年3月入団の新規団員募集
- **ターゲット**: 札幌市内の未就学児〜高校生とその保護者
- **予算**: 10万円
- **主な媒体**: Meta広告（Instagram/Facebook）、Google検索広告

## コンテンツ
- [index.html](index.html): 広告戦略のサマリー（KPI、予算配分、LTV戦略など）

## 運用方針
- **LTV重視**: 早期入団による長期的な在籍を重視した入団特典の提供
- **地域密着**: 札幌エリアに特化したターゲティング
